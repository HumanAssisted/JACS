quickstart() creates a persistent agent with keys on disk. If `./jacs.config.json` already exists, it loads it; otherwise it creates a new agent. Agent, keys, and config are saved to `./jacs_data`, `./jacs_keys`, and `./jacs.config.json`. Set `JACS_PRIVATE_KEY_PASSWORD` before calling quickstart (required). CLI users can also set `JACS_PASSWORD_FILE` to a password file path; set exactly one explicit source.
