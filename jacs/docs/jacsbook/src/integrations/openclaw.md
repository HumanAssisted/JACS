# OpenClaw Integration

This chapter has been reduced to a status note because the JACS repo does not currently contain a maintained OpenClaw integration surface.

The previous version of this page described a plugin workflow as if it were part of the supported JACS book path. That is too speculative for current documentation.

## Current Guidance

- If your real deployment is Express, Koa, FastAPI, LangChain, MCP, or A2A, use those sections of the book directly.
- Do not treat this repo as shipping a first-class OpenClaw integration today.

If OpenClaw becomes an actively maintained integration again, it should return with:

- a public implementation location
- example code that runs against the current APIs
- a small, concrete setup guide instead of a speculative platform chapter
